## Alarm Clock

Вашему вниманию представлена Максимально кривая, костыльная, убогая версия будильника которую вы видели.

**Кривее только ноги сына Уолтера Уайта из сериала Во все тяжкие**

### Для начала, как это работает.

1. Пользователь выбирает рингтон из предложенных
2. Ставит будильник на конкретное время **HH:MM**
3. Пишет сообщение которое он хочет получить вместе со звоном в ушах
4. Радуется, что пора спать (для этого я его делал)

## Рассмотрим 2 случая для времени 20:47:

### Случай 1: _Хочу уснуть пораньше_
```
Enter time (HH:MM): 23:55
Enter message: Time to sleep, sunshine!
```

- Строка **23:55** разбивается на подстроки и конвертируется в int и переводится в минуты
- Считаем текущее время в минутах
- Сравниваем с **target**-временем которое мы тоже перевели в минуты

Не трудно догадаться что происходит дальше

### Случай 2: _Посижу подольше_
```
Enter time (HH:MM): 01:40
Enter message: Time to sleep, sunshine!
```

- Строка **01:40** *так же* разбивается на подстроки и конвертируется в int и *так же* переводится в минуты
- *Так же* Считаем текущее время в минутах
- Сравниваем с **target**-временем которое мы тоже перевели в минуты

Так как target-время в переводе в минуты ну явно меньше чем текущее, в ход вступает флаг **tomorrow**

- Если текущее время почему то больше чем время до звонка, а target = false, то таргет меняет свое значение на true, указывая что звенеть надо завтра.
- В конце вы сдвигаем
